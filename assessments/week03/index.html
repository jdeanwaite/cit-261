Object Creation Functions

Objects in javascript are flexible and fun to make. An example of a built-in object you can create is the Date object

            
var date = new Date();

This will create a new date with the current date and time. Creating objects, giving them properties and methods, inheritance, and instantiation will be explored in this week.

Properties

Object properties are typically assigned with the dot notation, but also can be assigned using the bracket notation (like an associative array). These properties can be defined before the object is created, or added onto the object after creation.

            
function Car(year, make, model, color) {
    this.year = year;
    this.make = make;
    this.model = model;
    this.color = color;
    this.totalled = false;
}

Methods

Methods are functions that act on an object and are attached like properties. Typically the purpose of a method is to alter the state of an object or get a value from it.

            
Car.prototype.changeColor = function (color) {
    this.color = color;
};

Car.prototype.crashCar = function () {
    this.totalled = true;
};

Instantiation

An object is instantiated using the constructor defined when the object was defined.

            
function Car(year, make, model, color) {
    this.year = year;
    this.make = make;
    this.model = model;
    this.color = color;
    this.totalled = false;
}

var car = new Car(2005, "Nissan", "Frontier", "White");
console.log("car:", car.year, car.make, car.model);

Inheritance

Inheritance is the ability to extend the functionality of one object into another. It usually is a logical relationship. For example: a sports car is a car, but a car is not necessarily a sports car. Therefore, a sports car has all of the properties of a car, but with some extended functionality, such as the ability to "burn out" as in this example. We also note the top speed for a sports car:

            
function SportsCar(year, make, model, color, topSpeed) {
    Car.call(this, year, make, model, color);
    this.topSpeed = topSpeed;
}

SportsCar.prototype = Object.create(Car.prototype);
SportsCar.prototype.constructor = SportsCar;
SportsCar.prototype.burnOut = function() {
    if (this.totalled) {
        console.log("Sorry, no can do. This car is toast.");
    }
    else
    {
        console.log("Screeching noises and stuff.");
    }
};

var sportsCar = new SportsCar(2005, "Nissan", "GTR", "Blue");
console.log("sportsCar:", sportsCar.year, sportsCar.make, sportsCar.model);

console.log("sportsCar color:", sportsCar.color);
sportsCar.changeColor("blue");
console.log("sportsCar color:", sportsCar.color);

sportsCar.burnOut();
console.log("sportsCar totalled:", sportsCar.totalled);
sportsCar.crashCar();
console.log("sportsCar totalled:", sportsCar.totalled);
sportsCar.burnOut();
