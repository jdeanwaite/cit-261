Week 04

Week 04 includes the following assessments:

  • JSON Parse
  • JSON Stringify
  • Using XMLHTTPRequest to Consume a JSON Web Service

JSON Parse

JSON.parse() allows you to convert a string of JSON into a JavaScript object, provided the string is valid JSON. To test this functionality, type a valid JSON string into the textarea and click "Parse" to parse it and verify that it is valid JSON.




JSON Stringify

JSON.stringify works the opposite of JSON.parse in that it takes a JavaScript object and converts it into a string that can be used to send to a web service, store locally, or store in a file somewhere.

Click "Show me" to see an object stringified.


Using XMLHTTPRequest to Consume a JSON Web Service

Because JSON is so easy to convert between plain text and a JavaScript object, it is very powerful for use with RESTful API services. It makes data transfer easy and universal without the need to tediously parse XML.

Combining JSON with AJAX (XMLHTTPRequest) allows for an awesome user experience. AJAX allows the browser to load up content from another location without the need to reload the page.
